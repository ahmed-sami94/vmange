# Virtual Machines

The VM list shows live VirtualBox inventory reported by each host agent.

## Expanded details
Open one VM row to reveal Summary, Power, Snapshots, Storage, Network, Settings, Console, and Logs sections. Opening another VM closes the previous one.

## Actions
- Use the primary row action for the most common power action.
- Use the expanded sections for snapshots, storage, networking, and settings.
- Console in shared-hosting mode provides VRDE connection guidance and screenshot capture.

## Status
Runtime state is derived from the agent heartbeat. If a VM is running on the host, the next fresh inventory should show it as running in VMange.
