# Getting Started

VMange manages Linux hosts through outbound agents. Add a host, install the generated agent command on that machine, then use the dashboard for health, VirtualBox, Docker, scripts, and audit history.

## First steps
- Sign in with an administrator account.
- Add a host from Overview or Hosts.
- Run the generated install command on the target machine.
- Wait for the first heartbeat, then open the host page.

## Main areas
- Hosts show health, capabilities, uptime, and maintenance tools.
- Virtual Machines show live VirtualBox inventory and actions.
- Containers And Compose show Docker resources and saved stacks.
- Audit records queued actions and their results.
- Alarms track host thresholds and active notifications.
