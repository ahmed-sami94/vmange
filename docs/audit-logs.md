# Audit And Logs

Audit gives a record of dashboard actions, while operation logs show command progress and results.

## Status meanings
- `done` means the agent completed the action successfully.
- `failed` shows the most useful server or agent error available.
- `pending` means queued but not yet claimed by the agent.
- `running` means the agent is working on it.
- `expired` means it aged out before completion.

## Reading results
Hover or focus a status badge on desktop to see the reason. On mobile, tap the info hint beside the badge to reveal the same message.

Use the Alarms page for threshold policy history and current notifications.
