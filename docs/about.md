# About VMange

VMange is a free and open-source infrastructure management dashboard for Linux hosts, VirtualBox, Docker, Compose, scripts, terminals, monitoring, and alarms.

## Author
VMange is created by Ahmed Sami Abdelhamed, a cloud and infrastructure engineer with experience across system administration, cloud operations, virtualization, Linux, Docker, and service reliability.

## Contributing
VMange is released under Apache-2.0. Contributions, ideas, and bug reports are welcome.
