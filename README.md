# VMange

VMange is a free and open-source infrastructure management dashboard for Linux hosts, VirtualBox, Docker, Compose stacks, scripts, terminal workflows, monitoring, and alarms.

## Features
- Outbound host agents for shared-hosting-friendly management
- VirtualBox inventory and remote actions
- Docker containers, images, and saved Compose stacks
- Host monitoring dashboards and alarm policies
- Audited scripts and terminal commands
- Role-based access control, CSRF protection, audit logs, and per-host tokens

## Deployment
VMange supports:
- shared hosting subfolders
- subdomains
- standalone PHP hosts
- Docker deployments
- local/private VM deployments

See `README_DEPLOYMENT.md` and the authenticated Docs section after installation.
